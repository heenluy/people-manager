### Dependencies you need to run it
* Java 11 and openJDK
* Maven Apache
* NodeJs (the version used was 14.18.1)
* Angular 12

### Spring Dependencies
* Spring Data JPA
* Spring Web
* MySql Connector
* Flyway
* Validation
* Lombok

### Angular Dependencies
* Material UI
* Material Schematics
